# community
spring boot实现的学习社区
本项目主要实现了博客管理、论坛管理、赏币管理。用户对于博客可以打赏、评论、点赞、评论点赞，用户对于帖子可以跟帖回复、对跟帖点赞、被发帖者采纳获得金币，用户可以通过支付宝充值获得金币。在用户个人空间里面用户可以关注他人以及参看他人的一些信息。项目已上线。
